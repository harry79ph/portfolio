About

I have been working as a freelance photographer and using Photoshop for photo editing since 2014. My first website building attempt happened in 2016 when I realised I was no longer happy with the website, HE Photography that I had at that time.
As a computer geek, I could not help myself giving it a shot. I found a nice template and spent months on it trying to edit the codes and images to create the website that can suit my taste which you can find here. I re-designed this website again from scratch recently and you can find it in Projects section.

I also did some graphic design with Photoshop. When I decided to go into coding path with Java in 2018, the language I was advised to start with to understand programming concepts better, it became more of a passion and I realized that programming is what I really want to do. Then I learned Selenium which is a nice testing framework. Then I decided to go forward, do something more creative and started learning more web development technologies. So JavaScript became and still is my main focus. I would like to keep this journey learning new coding skills and expanding on the skills I already have.
Now I feel that I am ready to be a part of friendly team that we can strive together.

I think JavaScript is a great language and more fun than Java. I love Bootstrap and Sass as well, they are awesome tools to use on projects. And so far I have been enjoying using React and its component system is a great addition to JavaScript. During the learning process, I have designed many projects using those technologies and I have learned a lot from them. Main web development technologies I specialize in are:

  • HTML5
  • CSS3
  • Sass
  • Bootstrap
  • JavaScript
  • React
